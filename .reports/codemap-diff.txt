Citation Checker Implementation Details:
- check_credibility: Offline scoring via domain set lookup (24 research/news domains,
  8-item TLD score dict, 3 low-credibility domain host lists, URL shortener filter)
  Returns: {url, score 0.0-0.9, label "high"|"medium"|"low", reason string}
- check_reachability: HTTP HEAD request with follow_redirects=True, timeout=10s,
  latency tracking via time.monotonic()
  Returns: {url, reachable bool, status_code, latency_ms, final_url} (null if unreachable)
- Server: FastMCP("citation-checker-tool"), runs on 0.0.0.0:9004
- Tests: 27 unit tests + 1 live integration test (respx mocks httpx in unit suite)
